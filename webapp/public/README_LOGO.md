# Logo de FirstFrame

## Instrucciones para agregar el logo

1. **Coloca tu archivo de logo** en este directorio (`webapp/public/`) con el nombre:
   - `logo.png` (recomendado) o
   - `logo.svg` (si prefieres vectorial)

2. **Tamaño recomendado:**
   - Mínimo: 512x512px
   - Formato: PNG con fondo transparente o SVG

3. **El logo se mostrará automáticamente en:**
   - La página de inicio (Home)
   - El favicon del navegador
   - El icono de la aplicación en móviles

4. **Si no agregas el logo:**
   - Se ocultará automáticamente (no mostrará error)
   - La aplicación funcionará normalmente

## Estilos aplicados

El logo tiene una animación de brillo (glow) que hace un efecto neon suave,
coordinado con los colores morado y verde lila del tema de FirstFrame.
